# Log Investigation 1

## Objective
Analyze suspicious network activity and determine potential malicious behavior.

## Tools Used
- TryHackMe Lab
- Packet analysis
- Log review

## Steps Taken
1. Reviewed provided logs.
2. Identified unusual IP address activity.
3. Checked for abnormal port usage.
4. Correlated timestamps.

## Findings
Observed repeated connection attempts from a suspicious external IP address targeting multiple ports.

## Conclusion
This activity suggests possible reconnaissance or brute force attempts.
