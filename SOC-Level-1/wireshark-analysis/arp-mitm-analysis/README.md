# ARP Spoofing / MITM Analysis

## Overview
This case study focuses on detecting ARP poisoning (Man-in-the-Middle attack) using Wireshark. The goal is to analyze ARP traffic, identify abnormal behavior, and detect possible credential interception.

## Skills Demonstrated
- ARP traffic analysis
- Detection of spoofed MAC/IP mappings
- Identification of MITM attack patterns
- Network traffic interception analysis

## Tools Used
- Wireshark

## Notes
All analysis was performed in a controlled lab environment (TryHackMe).
